## 요구사항

### 레이아웃

- 헤더는 **상단 고정(sticky)** 으로 스크롤 시에도 항상 노출된다
- 배치: 좌측 로고 → 중앙 GNB 탭 → 우측 검색

### 1. 로고

- 토스증권 로고(심볼 + 텍스트)를 클릭하면 홈(`/`) 페이지로 이동한다
- 항상 헤더 좌측에 고정 노출된다

### 2. 탭 (GNB)

- 홈, 피드, 주식 골라보기, 내 계좌 총 4개 탭으로 구성된다
- 각 탭 클릭 시 해당 페이지로 이동한다
- 현재 활성화된 탭은 볼드(bold) 처리되어 강조 표시된다

| 탭 이름 | 경로 | 비고 |
|---------|------|------|
| 홈 | `/` | |
| 피드 | `/feed/recommended` | 1차에서는 빈 화면 |
| 주식 골라보기 | `/screener` | 1차에서는 빈 화면 |
| 내 계좌 | `/account` | 클릭 시 아무 반응 없음 (미구현) |

### 3. 검색

- 검색창 클릭 시 검색 입력이 활성화된다
- 키보드 `/` 단축키로 검색창을 포커스할 수 있다
- 플레이스홀더 텍스트: `/ 를 눌러 검색하세요`
- 종목명 또는 키워드 입력 시 **실시간 자동완성** 방식으로 드롭다운에 검색 결과를 표시한다
- 드롭다운 항목에는 **종목명만** 표시한다 (현재가/등락률은 추후 추가 가능)
- 드롭다운 항목 클릭 시 해당 종목 페이지로 이동한다

### 4. 로그인 버튼

- **1차 범위에서 제외** (로그인 UI 미고려)

## 예외 처리

| 상황 | 처리 방식 |
|------|-----------|
| 검색 결과가 없을 때 | `검색 결과 없음` 텍스트 표시 |
| 검색 API 에러 발생 시 | `검색 중 문제가 발생했습니다.` 텍스트 표시 |
| 내 계좌 탭 클릭 시 | 아무 반응 없음 (미구현) |

## 인터랙션 요약

| 트리거 | 동작 |
|--------|------|
| 로고 클릭 | 홈(`/`)으로 이동 |
| GNB 탭 클릭 | 해당 페이지로 이동, 활성 탭 볼드 처리 |
| 내 계좌 탭 클릭 | 아무 반응 없음 |
| 검색창 클릭 | 검색 입력 활성화 |
| 키보드 `/` 입력 | 검색창 포커스 |
| 검색어 입력 | 드롭다운으로 종목 검색 결과 표시 |
| 검색 결과 항목 클릭 | 해당 종목 페이지로 이동 |
| `Esc` 키 입력 | 검색 결과 삭제 (검색창 포커스는 유지) |
| 검색창 바깥 영역 클릭 | 검색창 닫기 |
