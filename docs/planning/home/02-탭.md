# 홈 - 차트 섹션 탭

## 개요

홈 화면 차트 섹션 상단에서 리스트의 "관점"을 전환하기 위한 섹션 내부 탭. 이번 범위에서는 모든 탭이 동일한 결과(실시간 차트)를 보여주며, 탭 자체는 활성 상태 시각 변경만 담당한다.

## 사용자 시나리오

1. 사용자가 홈(/)에 진입하면 차트 섹션의 첫 줄에 `실시간 차트` / `지금 뜨는 카테고리` / `국내 투자자 동향` 3개 탭이 노출되며, 가장 좌측인 `실시간 차트`가 활성 상태이다
2. 사용자가 다른 탭을 클릭하면 활성 탭이 바뀌고 볼드 처리되며 하단 바가 활성 탭 위치로 이동한다
3. 탭 변경에 따라 페이지 이동은 발생하지 않고, 같은 화면 내에서 활성 상태만 변경된다

## 화면 구성

### 화면 1: 탭

- 구성: `실시간 차트`, `지금 뜨는 카테고리`, `국내 투자자 동향` 총 3개
- 기본값: 가장 좌측인 `실시간 차트` 활성
- 활성 탭은 볼드 처리되며 하단 바가 활성 탭 위치로 이동한다
- 클릭 시 다른 페이지로 이동하지 않고 같은 화면 내에서 활성 탭만 변경된다

## 데이터 요구사항

별도 데이터 없음. 활성 탭 ID(클라이언트 상태)만 보유한다.

## 비즈니스 규칙

- 기본 활성 탭은 가장 좌측 항목(`실시간 차트`)이다
- **이번 범위 한정 규칙**: 어떤 탭을 클릭해도 하단 리스트는 항상 `실시간 차트` 결과로 동일하게 유지한다 (탭 변경이 [03-필터.md](./03-필터.md) / [04-리스트.md](./04-리스트.md)에 영향 없음)
