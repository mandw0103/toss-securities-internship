## 개요

홈 차트 섹션의 정렬·필터 컨트롤. 시장 / 정렬 기준 / 기간 세 그룹으로 구성되며, 각 그룹은 항상 정확히 1개만 선택된다. 변경 시 [04-리스트.md](./04-리스트.md)가 즉시 재구성된다.

## 컴포넌트 구조

```
ChartSection                       # 02-탭 스펙에서 정의된 wrapper
└── ChartSectionFilter              # 본 스펙
    ├── FilterGroup (시장)
    │   ├── FilterPill × 3
    │   └── SlidingIndicator
    ├── FilterGroup (정렬)
    │   ├── FilterPill × 6
    │   └── SlidingIndicator
    └── FilterGroup (기간)
        ├── FilterPill × 7
        └── SlidingIndicator
```

## 타입 정의

```ts
// src/types/chart-filter.ts
import type { Period, Market } from '@/types/stock';

export type MarketFilter = '전체' | Market;

export type SortKey =
  | '토스증권 거래대금'
  | '토스증권 거래량'
  | '거래대금'
  | '거래량'
  | '급상승'
  | '급하락';

export interface ChartFilterState {
  market: MarketFilter;
  sort: SortKey;
  period: Period;
}

export const MARKET_OPTIONS: MarketFilter[] = ['전체', '국내', '해외'];

export const SORT_OPTIONS: SortKey[] = [
  '토스증권 거래대금',
  '토스증권 거래량',
  '거래대금',
  '거래량',
  '급상승',
  '급하락',
];

export const PERIOD_OPTIONS: Period[] = [
  '실시간', '1일', '1주일', '1개월', '3개월', '6개월', '1년',
];

export const DEFAULT_FILTER: ChartFilterState = {
  market: '전체',
  sort: '토스증권 거래대금',
  period: '실시간',
};
```

`Period` 타입은 [common/02-주식데이터.md](../common/02-주식데이터.md)의 정의를 그대로 사용한다.

## 컴포넌트 명세

### ChartSectionFilter

| 속성 | 타입 | 필수 | 설명 |
|------|------|------|------|
| value | ChartFilterState | O | 현재 필터 값 |
| onChange | `(next: ChartFilterState) => void` | O | 변경 콜백 |

**동작:**
- 세 `FilterGroup`을 가로로 배치 (시장 → 정렬 → 기간).
- 각 그룹의 `onSelect`에서 해당 필드만 갱신한 새 객체로 `onChange`를 호출.

### FilterGroup

| 속성 | 타입 | 필수 | 설명 |
|------|------|------|------|
| options | string[] | O | 항목 목록 |
| value | string | O | 현재 선택값 |
| onSelect | `(value: string) => void` | O | 선택 콜백 |

**동작:**
- `options`를 가로로 나열.
- 선택된 항목은 볼드 처리.
- `SlidingIndicator`(흰 박스)를 선택된 항목의 위치로 이동.

### FilterPill

| 속성 | 타입 | 필수 | 설명 |
|------|------|------|------|
| label | string | O | 표시 텍스트 |
| isActive | boolean | O | 활성 여부 |
| onClick | `() => void` | O | 클릭 콜백 |

**동작:**
- 클릭 시 `onClick` 호출.
- `isActive=true`이면 볼드.

### SlidingIndicator

| 속성 | 타입 | 필수 | 설명 |
|------|------|------|------|
| left | number | O | 활성 항목의 좌측 오프셋(px) |
| width | number | O | 활성 항목의 너비(px) |

**동작:**
- 부모 `FilterGroup`이 `useLayoutEffect`로 활성 pill의 `offsetLeft`/`offsetWidth`를 측정해 props로 전달한다.
- 본 컴포넌트는 받은 `left`/`width`로 가로 transform 슬라이드 애니메이션을 재생한다.

> 필터 항목 텍스트 길이가 가변(`전체` vs `토스증권 거래대금`)이라 인덱스만으로는 흰 박스의 너비/위치를 구할 수 없다. 따라서 인덱스 기반이 아니라 픽셀 단위로 직접 받는다.

## 상태 관리

### 클라이언트 상태

- [02-탭.md](./02-탭.md)의 `ChartSection` 부모 컴포넌트에서 `useState<ChartFilterState>(DEFAULT_FILTER)`로 보유.
- 본 컴포넌트는 props로 `value`/`onChange`를 받기만 하고, 동일 부모를 공유하는 [04-리스트.md](./04-리스트.md)로 같은 값이 전달된다.
- 세 그룹 중 하나라도 변경되면 즉시 [04-리스트.md](./04-리스트.md)가 재정렬·재필터링된다.
- "투자위험 주식 숨기기" 토글은 이번 범위에서 의도적으로 제외 (기획서 명시).
