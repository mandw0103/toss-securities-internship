## 개요

홈 화면 차트 섹션 상단의 3개 탭. 활성 탭의 시각 변경(볼드 + 슬라이딩 하단 바)만 담당하며, 본 범위에서는 [03-필터.md](./03-필터.md)·[04-리스트.md](./04-리스트.md)에 영향을 주지 않는다.

## 컴포넌트 구조

```
ChartSection                       # 02/03/04 공통 부모 (본 스펙에서 정의)
├── ChartSectionTabs                # 본 스펙
│   ├── TabItem × 3
│   └── TabUnderline
├── ChartSectionFilter              # 03 스펙
└── ChartSectionList                # 04 스펙
```

## 컴포넌트 명세

### ChartSection

홈 차트 섹션의 wrapper. 02/03/04 스펙이 공유하는 클라이언트 상태(활성 탭 + 필터)를 보유한다.

| 속성 | 타입 | 필수 | 설명 |
|------|------|------|------|
| stocks | Stock[] | O | [01-지표.md](./01-지표.md)의 HomePage에서 `useStocks()` 결과를 그대로 전달받음 |

**내부 상태:**
- `activeTab: ChartTab` — 활성 탭 (기본값 `'실시간 차트'`)
- `filter: ChartFilterState` — 필터 ([03-필터.md](./03-필터.md))

**동작:**
- `<ChartSectionTabs>`에 `activeTab`과 `setActiveTab`을 전달.
- `<ChartSectionFilter>`에 `filter`와 `setFilter`를 전달.
- `<ChartSectionList>`에 `stocks`와 `filter`를 전달.

> `useStocks()`는 본 컴포넌트가 직접 호출하지 않는다. HomePage가 1회 호출 후 props로 내려보낸다.

### ChartSectionTabs

| 속성 | 타입 | 필수 | 설명 |
|------|------|------|------|
| activeTab | ChartTab | O | 현재 활성 탭 |
| onChange | `(tab: ChartTab) => void` | O | 탭 변경 콜백 |

**동작:**
- 3개 탭을 좌→우로 나열.
- `activeTab`과 일치하는 항목을 볼드 처리.
- `TabUnderline`을 활성 탭의 가로 위치로 슬라이드 이동.

**탭 데이터:**

```ts
// src/types/chart-tab.ts
export type ChartTab = '실시간 차트' | '지금 뜨는 카테고리' | '국내 투자자 동향';

export const CHART_TABS: ChartTab[] = [
  '실시간 차트',
  '지금 뜨는 카테고리',
  '국내 투자자 동향',
];
```

### TabItem

| 속성 | 타입 | 필수 | 설명 |
|------|------|------|------|
| label | ChartTab | O | 탭 텍스트 |
| isActive | boolean | O | 활성 여부 |
| onClick | `() => void` | O | 클릭 콜백 |

**동작:**
- 클릭 시 `onClick` 호출.
- `isActive=true`이면 볼드.
- 페이지 이동 없음.

### TabUnderline

| 속성 | 타입 | 필수 | 설명 |
|------|------|------|------|
| activeIndex | number | O | 활성 탭 인덱스 (0~2) |

**동작:** 활성 탭 위치로 가로 transform 슬라이드 애니메이션.

## 상태 관리

### 클라이언트 상태

- `ChartSection` 부모 컴포넌트에서 `useState<ChartTab>('실시간 차트')`로 보유.
- 기본값: 가장 좌측 항목인 `'실시간 차트'`.
- **이번 범위 한정**: `activeTab` 값은 [03-필터.md](./03-필터.md)·[04-리스트.md](./04-리스트.md) 동작에 영향을 주지 않는다 (어떤 탭이든 리스트는 항상 `실시간 차트` 결과로 동일).
